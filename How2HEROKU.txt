Title: Deploying App in Heroku and Git
#################################################
1. Prepair your App for Heroku
	- App-Folder
		- view-Folder
		- app.rb
		- config.ru (require "./app"ENTER run Sinatra::Application
		- Gemfile (source "https://rubygems.org" ENTER gem "sinatra" etc., needed gems here)
		- README.md

2. Install all libraries in App-Folder
   cmd>> bundle install
	- creates 
		- Gemfile.lock

3. Login to heroku with:
   cmd>> heroku login

##########   Add to Git!!! ######################

4. Before u can push an app to heroku u need to initialize a local Git repository and commit your files
     Create git repository in my app's directory
   cmd/sinatra>> git init
   cmd/sindatra>> git add FOLDERNAME/* #adds foldr 2 REPO
       ### Test on : git status ####

   cmd/sinatra>> git commit -m "Added Folder"
   #### Also for making local changes #####

   cmd/sinatra>> git remote add REMOTENAME <URL2REPO>
   cmd/sinatra>> git push -u REMOTENAME master

	Login
##################################################

5. Add all files to git repository with:
   cmd>> git add .

6. Commit (update version) your files with
   cmd>> git commit -m "Initial commit"
  
############# Add to Heroku ###################

7. Create a heroku app in my app directory
   cmd>> heroku create

8. Deploy your app with:
   cmd>> git push heroku master

9. Open your app in your browser:
   cmd>> heroku open

########### Additonal Commands ################

xx. Zur letzten Version zurück mit:
    cmd>> git reset --hard

xx. Copy of remote repository:
    cmd>> git clone >>URL<<

xx. Updating from remote with:
    cmd>> git pull

xx. Updating local changes to remote:
    cmd>> git push

xx. See changes made
    cmd>> git diff <filname>
    cmd>> Output of Changes in Working progress ready to stage
    cmd>> q # to quit diff mode
